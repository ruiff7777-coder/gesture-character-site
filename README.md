# 共感 · 形象交互

一个完全在浏览器本地运行的卡通角色手势互动网站。

## 运行

摄像头 API 需要 localhost 或 HTTPS。进入项目目录后运行：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。

不要直接双击 `index.html` 用 `file://` 打开。不同浏览器会限制本地模块、摄像头权限或模型加载，可能出现按钮无响应、摄像头打不开、一直加载中的情况。

Codex 内置浏览器适合检查页面是否加载，但通常不能稳定接入你的真实摄像头。测试手势请使用本机默认浏览器访问 `http://localhost:4173`，并允许摄像头权限。

## 手势

- 挥手：角色微笑。
- 握拳保持约 0.6 秒：角色进入睡眠。
- 睡眠时张开五指保持约 0.6 秒：角色苏醒。

摄像头画面只在当前浏览器内处理，不会上传。当前版本使用 MediaPipe 手部关键点模型定位五个指尖点：五个小点同步大幅移动会触发挥手，一个大点会触发握拳，五个小点会触发张掌。为了便于 GitHub Pages 部署，手势识别的 WASM 和模型文件通过官方 HTTPS 地址加载。为了避免脸部肤色误判，模型不可用时不会再使用肤色方框猜测手势。无摄像头权限时仍可通过底部演示按钮体验全部状态。
